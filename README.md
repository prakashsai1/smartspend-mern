# 💸 SmartSpend MERN App

A full-stack expense tracker with login and charts.

## 🚀 Features
- Add, Edit, Delete expenses
- Category-wise chart
- Login authentication
- Clean UI

## 🛠 Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

## ▶️ Run Locally

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm start
